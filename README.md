## Get Started

1. In the terminal run `npx expo start`.
1. To see the app on mobile, scan QR code with `Expo Go` app.
1. To see the web app in action, press `w` in the terminal. It will open the web app in the default web browser.

## Tech Stack

- React Native
- Expo
- TypeScript
- AsyncStorage
